## 🧭 About

This is the **third official version** of the website for **SEDS Antariksh – the Space Club of VIT Chennai**, designed to represent our mission, projects, and events.

The website includes:

- Club introduction & vision  
- 3D interactive elements powered by **Three.js**  
- Details about our events, teams, and technical projects  
- Blog & newsletter signup  
- Fully responsive and animated UI

## 🛠️ Getting Started

### 📦 Prerequisites

Make sure you have:

- [Node.js](https://nodejs.org) (v14 or newer)  
- npm or yarn package manager

### 🔧 Installation

```bash
git clone https://github.com/SEDS-Antariksh-VIT-Chennai/official-website-v3.git
cd official-website-v3
npm install
```

### 🧪 Run Locally

```bash
npm start
# or
yarn dev
```

Visit `http://localhost:3000` to view it live.

### 🏗️ Build for Production

```bash
npm run build
```
